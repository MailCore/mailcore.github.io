Overview

This is not a class you instantiate! It has only two class methods, and that is all you need to send e-mail. First use CTCoreMessage to compose an e-mail and then pass the e-mail to the method sendMessage: with the necessary server settings and CTSMTPConnection will send the message.

canConnectToServer:username:password:port:connectionType:useAuth:error:

Use this method to test the user’s credentials.

+ (BOOL)canConnectToServer:(NSString *)server username:(NSString *)username password:(NSString *)password port:(unsigned int)port connectionType:(CTSMTPConnectionType)connectionType useAuth:(BOOL)auth error:(NSError **)error

Parameters

server

The server address

username

The username, if there is none then pass in an empty string. For some servers you may have to specify the username as username@domain

password

The password, if there is none then pass in an empty string.

port

The port to use, the standard port is 25

connectionType

What kind of connection, either: CTSMTPConnectionTypePlain, CTSMTPConnectionTypeStartTLS, CTSMTPConnectionTypeTLS

auth

Pass in YES if you would like to use SASL authentication

error

Will contain an error when the method returns NO

Return Value

Returns YES on success, NO on error

Discussion

Use this method to test the user’s credentials.

This is useful for account setup. You can have the user enter in their credentials and then verify they work without sending a message.
