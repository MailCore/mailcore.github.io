Overview

CTCoreAccount is the base class with which you establish a connection to the IMAP server. After establishing a connection with CTCoreAccount you can access all of the folders (I use the term folder instead of mailbox) on the server.

connectToServer:port:connectionType:authType:login:password:

This method initiates the connection to the server.

- (BOOL)connectToServer:(NSString *)server port:(int)port connectionType:(int)conType authType:(int)authType login:(NSString *)login password:(NSString *)password

Parameters

server

The address of the server.

port

The port to connect to.

authType

The authentication type, only CTImapAuthTypePlain is currently supported

login

The username to connect with.

password

The password to use to connect.

connnectionType

What kind of connection to use, it can be one of these three values: CTConnectionTypePlain, CTConnectionTypeStartTLS, CTConnectionTypeTryStartTLS, CTConnectionTypeTLS. See MailCoreTypes.h for more information

Return Value

Return YES on success, NO on error. Call method lastError to get error if one occurred

Discussion

This method initiates the connection to the server.
